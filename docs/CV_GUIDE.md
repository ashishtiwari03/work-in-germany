# 📄 German CV (Lebenslauf) & Cover Letter (Anschreiben) Guide

> **Everything you need to know about applying for jobs in Germany** — from format to common mistakes to what recruiters actually look for.

## 🇩🇪 How German Applications Differ

If you're coming from the US, UK, India, or most other countries, German job applications will feel very different. Here's the core mindset shift:

| What you're used to | What Germany expects |
|---------------------|---------------------|
| 1-page resume | 2-page **tabular** CV (Lebenslauf) |
| No photo | **Photo expected** (professional headshot) |
| Generic cover letter | **Tailored Anschreiben** for each application |
| Soft skills emphasis | **Qualifications & certificates** emphasis |
| Casual/creative formats | **Conservative, structured** format |
| References on request | **Arbeitszeugnis** (employment references) attached |
| Self-promotion is good | **Understated competence** is preferred |

## 📝 The Lebenslauf (CV)

### Structure

A German CV is typically **reverse-chronological** and **tabular** (two-column layout). Here's the expected structure:

### 1. Personal Details (Persönliche Daten)
- Full name
- Address (city is enough, no full address needed for privacy)
- Phone number (with country code if international)
- Email address
- Date of birth *(optional but common — anti-discrimination laws apply, but many still include it)*
- Nationality *(optional but useful for visa context)*
- LinkedIn / GitHub *(increasingly common in tech)*
- **Professional photo** — top right corner

**About the photo:** Yes, it's expected. Despite anti-discrimination laws, ~90% of German applications include one. Get a professional headshot: neutral background, business casual attire, friendly but professional expression. Passport-style works. Do NOT use a selfie, vacation photo, or full-body shot.

### 2. Work Experience (Berufserfahrung)
- **Reverse chronological** — most recent first
- For each position: dates (MM/YYYY – MM/YYYY), company name, job title, 3-5 bullet points of responsibilities/achievements
- Include Werkstudent, internships, and working student positions — these count in Germany
- **No gaps**: Germans notice gaps. If you traveled, studied, or did a Sabbatical, mention it briefly

### 3. Education (Ausbildung / Studium)
- University degrees with grades (if good — German grading: 1.0 is best, 4.0 is passing)
- Thesis title if relevant
- For international degrees: mention the equivalent (e.g., "B.Sc. Computer Science — equivalent to German Bachelor")
- Abitur / High School only if you're a recent grad

### 4. Skills (Kenntnisse)
- **Languages**: German (B2), English (C1), etc. — use CEFR levels
- **Technical skills**: Programming languages, frameworks, tools
- Rate your proficiency honestly: Grundkenntnisse (basic), gut (good), sehr gut (very good), fließend (fluent)

### 5. Certifications & Training (Weiterbildung)
- Professional certificates (AWS, Azure, Scrum, etc.)
- Relevant online courses or bootcamps
- Germans value formal qualifications

### 6. Volunteer Work / Hobbies (Ehrenamt / Hobbys)
- Optional but valued, especially if relevant
- Shows personality and Teamfähigkeit (team skills)

---

### Common Mistakes to Avoid

1. **US-style resume**: One page, no photo, creative layout → won't work for traditional German companies. Tech startups are more flexible.
2. **Gaps without explanation**: A 6-month gap will be questioned. Add "Career break — personal development" or similar.
3. **Lying about German skills**: If you claim B2 and can't hold a basic conversation, the interview will be awkward.
4. **Wrong date format**: Use DD.MM.YYYY or MM/YYYY — never MM/DD/YYYY.
5. **Missing signatures**: Some traditional employers expect a handwritten signature and date at the bottom of the CV. This is becoming less common in tech but still exists.
6. **Sending as .docx**: Send as **PDF** unless explicitly asked for Word format.

## ✉️ The Anschreiben (Cover Letter)

The Anschreiben is **mandatory** for most German applications. Even if the job ad doesn't explicitly ask for one, include it.

### Structure (1 page, formal)

**Header:**
```
Your Name
Your Address
Your Phone | Your Email

City, DD.MM.YYYY

Company Name
Hiring Manager Name (if known)
Company Address

Betreff: Bewerbung als [Job Title] — Referenznummer [if any]
```

**Opening paragraph (2-3 sentences):**
- Why this company, why this role, why now
- Avoid generic openers like "Hiermit bewerbe ich mich..." (I hereby apply...)
- Show you've researched the company

**Middle paragraphs (2-3 paragraphs):**
- Your relevant qualifications and experience
- Concrete examples of achievements (with numbers if possible)
- Why you're a good fit for THIS specific role
- If you're international: address the elephant in the room — your visa situation and language skills

**Closing paragraph:**
- Availability and start date (Kündigungsfrist if currently employed)
- Salary expectation (Gehaltsvorstellung) — ONLY if the job ad asks for it
- Express enthusiasm, request an interview

**Sign-off:**
```
Mit freundlichen Grüßen
[Your Name]
```

### Tips for International Applicants

- **Write in the language of the job ad.** If the ad is in English, write in English. If in German, write in German.
- **Address visa proactively:** "I hold an EU Blue Card / My qualifications meet Blue Card requirements, and I can begin work immediately upon visa approval."
- **Don't apologize for your German:** Instead of "My German is not perfect," say "I am currently at B1 level and actively improving through intensive courses."
- **Research salary expectations:** Use [Glassdoor](https://www.glassdoor.de), [Kununu](https://www.kununu.com), [Levels.fyi](https://www.levels.fyi) for Germany, or [Gehalt.de](https://www.gehalt.de). Always state **Brutto** (gross) annual salary.

## 📎 The Complete Application Package (Bewerbungsunterlagen)

A full German application typically includes:

1. **Anschreiben** (cover letter) — 1 page PDF
2. **Lebenslauf** (CV) — 1-2 page PDF with photo
3. **Zeugnisse** (certificates):
   - University degree certificate (and transcript if available)
   - Arbeitszeugnisse (employment references from previous German employers)
   - Relevant certificates (language, professional)
4. **Optional:**
   - Portfolio link / GitHub
   - Recommendation letters

**Important:** Combine everything into a **single PDF** file (unless the portal asks for separate uploads). Name it: `Bewerbung_[YourName]_[JobTitle].pdf`

## 🏢 Startup vs. Traditional: Know Your Audience

| | Traditional (DAX, Mittelstand) | Startup / International |
|---|---|---|
| **CV format** | Tabular, formal, photo mandatory | Modern/creative OK, photo optional |
| **Anschreiben** | Mandatory, formal | Sometimes optional (check the ad) |
| **Language** | Often German | Usually English |
| **Salary in application** | State if asked | Often discussed later |
| **Application method** | Email or portal | Portal, LinkedIn Easy Apply |
